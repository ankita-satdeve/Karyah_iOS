# Karyah iOS

Karyah is a SwiftUI-based iOS application built using MVVM and Clean Architecture principles.

## Features

- SwiftUI
- MVVM Architecture
- Dependency Injection
- Core Data
- Async/Await Networking
- Repository Pattern
- Offline Support
- Dark Mode Support

## Tech Stack

- Swift 6
- SwiftUI
- Core Data
- URLSession
- Combine
- Xcode

## Project Structure

Karyah_iOS
├── App
├── Core
├── Data
├── Domain
├── Features
├── Resources
└── Tests

## Requirements

- iOS 17.0+
- Xcode 26+

## Installation

1. Clone the repository
2. Open `Karyah_iOS.xcodeproj`
3. Build and run

## Architecture

The project follows:

- Clean Architecture
- MVVM
- SOLID Principles
- Dependency Injection
